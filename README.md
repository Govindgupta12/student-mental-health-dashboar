# Student Mental Health Dashboard

This project is based on student mental health data analysis using SQL and Tableau.  
The main goal of the project was to clean the dataset, organize the data properly, and create a dashboard to understand different student-related factors visually.

The dashboard includes analysis of:
- Sleep Duration
- Study Hours
- Study Satisfaction
- Academic Pressure
- Financial Stress
- Depression Status

## Tools Used
- SQL Server
- Tableau
- Excel / CSV

## Work Done in SQL
- Cleaned and formatted the dataset
- Standardized values in columns
- Checked null and blank values
- Created age groups
- Converted data types where required
- Prepared the data for visualization

## Dashboard
The dashboard was created in Tableau to visualize student trends and patterns in a simple and interactive way.

## Files Included
- `student_dashboard.twbx` → Tableau dashboard file
- `SQLQuery1.sql` → SQL queries used in the project
- `dashboard_screenshot.png` → Dashboard screenshot
